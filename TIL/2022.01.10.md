### setState

State의 값을 바꿀 때에는  setState를 이용해야 된다.

```jsx
import React, { Component } from 'react';
import { View, Text, StyleSheet } from 'react-native';

class App extends Component {

  state = {
    sampleText: 'Hello World',
    sampleBoolean: false
  }

  changeState = () => {
    if (!this.state.sampleBoolean) {
      this.setState({
        sampleText: 'Text Changed!!!',
        sampleBoolean: true
      })
    } else {
      this.setState({
        sampleText: 'Hello World!!!',
        sampleBoolean: false
      })
    }
  }

  render() {
    return (
      <View style={styles.background}>
        <Text onPress={this.changeState}>
          {this.state.sampleText}
        </Text>
      </View>
    )
  }
}

const styles = StyleSheet.create({
  background: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center'
  }
})

export default App;
```

setState는 비동기적으로 실행됨. 즉, 한번에 state의 값을 바꿀수 없다. prevState를 이용해서 바꿀 수 있다

```jsx
import React, { Component } from 'react';
import { View, Text, StyleSheet } from 'react-native';

class App extends Component {

  state = {
    sampleText: 'Hello World',
    sampleBoolean: false,
    sampleNum: 1
  }

  onAdd = () => {
    this.setState(prevState => {
      return {
        sampleNum: prevState.sampleNum + 1
      }
    })
  }

  render() {
    return (
      <View style={styles.background}>
        <Text onPress={this.onAdd}>
          {this.state.sampleNum}
        </Text>
      </View>
    )
  }
}

const styles = StyleSheet.create({
  background: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center'
  }
})

export default App;
```

### Props

read only. 읽기전용 property. 부모가 쓴 내용을 자식에게 복사하는 개념. 자식은 수정 불가.

PropsChild.js

```jsx
import React, { Component } from 'react';
import { View, Text } from 'react-native';

const PropsChild = (props) => {
    return (
        <View>
            <Text onPress={props.changeState}>
                {props.sampleText}
            </Text>
        </View>
    )
}

export default PropsChild
```

App.js

```jsx
import React, { Component } from 'react';
import { View, Text, StyleSheet } from 'react-native';
import PropsChild from './propsChild';

class App extends Component {

  state = {
    sampleText: 'Hello World',
    sampleBoolean: false,
  }

  changeState = () => {
    if (!this.state.sampleBoolean) {
      this.setState({
        sampleText: 'Text Changed!!!',
        sampleBoolean: true
      })
    } else {
      this.setState({
        sampleText: 'Hello World!!!',
        sampleBoolean: false
      })
    }
  }

  render() {
    return (
      <View style={styles.background}>
        <PropsChild sampleText={this.state.sampleText} changeState={this.changeState}/>
      </View>
    )
  }
}

const styles = StyleSheet.create({
  background: {
    flex: 1,
    backgroundColor: '#fff',
    alignItems: 'center',
    justifyContent: 'center'
  }
})

export default App;
```
